題目敘述

438. Find All Anagrams in a String
給一字串 s ，在 s 中找尋任何排列組合的字串 p ，
有點難敘述，親自看過題目會比較清楚!

解題思路

這題用 sliding window 做，
一開始做題時忘記 sliding window 的特性，
跑簡單測資時都會對，測資一旦變複雜就會導致 runtime error。
在滑行 window 時不需要整個 window 重新載入，
只需要丟棄最前面的元素，並新增下一個元素就行了!!!!!
記起來了~

在比對時如果是 string 的話就需要先 sort 才能比對，
所以用 vector 建 hash table ，比較有效率!
